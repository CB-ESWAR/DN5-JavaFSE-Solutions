# Exercise 6 - Library Management System

## Objective

Implement searching techniques in a Library Management System.

## Algorithms Used

- Linear Search
- Binary Search

## Time Complexity

### Linear Search
- Best Case : O(1)
- Average Case : O(n)
- Worst Case : O(n)

### Binary Search
- Best Case : O(1)
- Average Case : O(log n)
- Worst Case : O(log n)

## Conclusion

Binary Search is more efficient than Linear Search for sorted data.
