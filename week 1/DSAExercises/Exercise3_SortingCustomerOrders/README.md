# Exercise 3 - Sorting Customer Orders

## Objective
Sort customer orders based on total price.

## Algorithms Used
- Bubble Sort
- Quick Sort

## Time Complexity

Bubble Sort
- Best Case : O(n)
- Average Case : O(n²)
- Worst Case : O(n²)

Quick Sort
- Best Case : O(n log n)
- Average Case : O(n log n)
- Worst Case : O(n²)

## Conclusion

Quick Sort is much faster than Bubble Sort for large datasets.
